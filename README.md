**📚 LinguaLearn AI**

LinguaLearn AI is an AI-powered multilingual learning assistant designed to help students understand NCERT science concepts through interactive explanations, chapter-based learning, and AI-generated quizzes.
The platform transforms traditional textbook learning into an interactive AI tutoring experience, allowing students to ask questions, explore chapters, and test their understanding instantly.

**🚀 Features**

🤖 AI Tutor
Ask questions about any chapter and get simple, clear explanations powered by AI.

📖 Chapter-Based Learning
Navigate through class, subject, and chapter to learn concepts aligned with the NCERT curriculum.

📝 AI Quiz Generator
Generate practice MCQs instantly to test understanding after learning a topic.

🌍 Multilingual Support
Students can learn in multiple languages, improving accessibility across regions.

💬 Interactive Chat
Students can ask unlimited doubts and receive instant explanations.

**⚙️ Installation**

1️⃣ Clone the repository
git clone https://github.com/yourusername/lingualearnai.git
cd lingualearnai
2️⃣ Install dependencies
npm install
3️⃣ Setup environment variables
Create a .env file:

ANTHROPIC_API_KEY=your_api_key_here
4️⃣ Run the backend server
node server.js
5️⃣ Start the React app
npm start
